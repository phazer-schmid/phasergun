# 21 CFR Part 807 - Establishment Registration and Device Listing for FDA

## Overview
Title 21 Code of Federal Regulations (CFR) Part 807 contains FDA regulations for:
- Establishment registration
- Device listing
- Premarket notification (510(k))

## Key Sections for 510(k) Submissions

### §807.87 - Contents of a premarket notification
**(a) Device name** - Including trade/proprietary name, common name, and classification name

**(b) Registration number** - If applicable

**(c) Class** - Device class per 21 CFR 860

**(d) Action taken** - Description of action taken to qualify for submission

**(e) **510(k) labeling** - Proposed labeling

**(f) Comparison with similar devices** - Table comparing device to predicate(s)

**(g) Performance data** - Including clinical data when required

**(h) Clinical investigations** - Summary of clinical studies if conducted

**(i) Financial disclosure** - If clinical studies conducted

**(k) Summary or statement** - Either 510(k) summary (§807.92) or statement (§807.93)

**(l) Truthful and accuracy statement** - Certification that submission is complete

### §807.92 - 510(k) Summary
Must include:
1. Submitter name and address
2. Trade/proprietary name, common name, classification name
3. Predicate device information
4. Device description
5. Substantial equivalence comparison
6. Performance data summary

### §807.93 - 510(k) Statement
Alternative to summary stating device is substantially equivalent

## Substantial Equivalence (SE) Requirements

Per §807.92, device is SE if:
1. Same intended use as predicate
2. Same technological characteristics OR
3. Different technological characteristics that don't raise different safety/effectiveness questions

## Key Definitions

**Predicate Device**: Legally marketed device to which subject device is compared

**Substantial Equivalence**: Device has same intended use and same/similar technological characteristics as predicate

**510(k) Number**: FDA clearance number (format: K######)

## Common RTA (Refuse to Accept) Reasons
- Incomplete device description
- Missing predicate comparison
- Inadequate performance data
- Missing labeling
- Incomplete 510(k) summary
